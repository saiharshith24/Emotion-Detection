# Emotion Detection using Machine Learning

This project detects emotions from text using NLP and machine learning.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF
- Logistic Regression

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python src/train.py

3. Run the application:
   python app.py
